## Deskripsi Tugas
Repositori ini berisi implementasi dasar JavaScript yang mencakup pendeklarasian variabel, penggunaan berbagai tipe data, serta penerapan operator aritmatika dan logika.

## Langkah Implementasi
* **Step 1:** Project Setup (Folder & `script.js`).
* **Step 2:** Implementasi variabel (`var`, `let`, `const`) dan menampilkannya ke DOM (HTML).
* **Step 3:** Eksperimen tipe data dan pengecekan menggunakan `typeof`.
* **Step 4:** Penerapan operator aritmatika dan logika kondisi (Studi Kasus: Izin Mendaki).
* **Step 5:** Validasi hasil melalui Browser Console.
